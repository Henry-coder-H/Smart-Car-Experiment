本仓库为智能工程学院 2026 学期智能车课程项目代码仓库，竞赛最终排名为 1st 😊。

## 项目简介

本项目主要围绕智能车路径规划、控制器参数调试、通信控制以及实验路线构建等内容展开。当前代码在比赛中已经能实现较好的效果。

## 待优化方向

代码现在还没有深度研究超参数之间的作用，而且还可以尝试改进控制器，以及进一步优化动态路径规划的算法等，欢迎大家在本仓库的基础上进行改进优化！😋😋😋

## 小tips

在程序违规的判定中，超速判定其实是25m/s，但是实验指导书上说的是20m/s，大家或许可以根据此调高一下自己小车的速度。但作者当时没有观察到这一点，所以小车的限速依旧是20m/s。😭
